Sơ Đồ Kiến Trúc Hệ Thống Smart Parking (Hoàn Chỉnh)

Tất cả luồng được nối với nhau, đầy đủ Triggers

Tóm tắt đầy đủ các Trigger:

  • Trigger 1: API Gateway → Lambda account-api (/login, /register, /root)
  • Trigger 2: Cognito Post Confirmation → Lambda smart-parking-save-user-to-db (lưu user vào DynamoDB)
  • Trigger 3: S3 → Lambda S3 Trigger (xử lý ảnh mới upload)
  • Trigger 4: DynamoDB Stream → Lambda DynamoDB Stream (cập nhật analytics khi dữ liệu thay đổi)
  • Trigger 5: IoT Core Rule → Lambda IoT Rule (xử lý tin nhắn MQTT từ thiết bị IoT)
  • Trigger 6: API Gateway → Lambda parking-api (/parking/*, /vehicle/*, /analytics/*, /ai/chat)
  • Tất cả luồng đều được nối với nhau liên tục!
